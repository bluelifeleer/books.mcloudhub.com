### Python中使用算术运算符

#### Python中的算术运算符

|  运算符  |  描述  |
|----|----|
|  +  |  加法  |
|  -  |  减法   |
|  *  |  乘法  |
|  %  |  取模  |
|  /  |  除法  |
|  //  |  整除  |
|  **  |  求幂  |
|  &  |  二进制与  |
|  1  |  二进制或  |
|  ^  |  按位异或  |
|  ~  |  二进制补码  |
|  <<  |  向左移位  |
|  >>  |  向右移位  |
|  and  |  逻辑与  |
|  or  |  逻辑或  |
|  not  |  逻辑非  |

###### 注：1表示"|"。

#### 提升运算符优先级

> 在python中运算符遵循数学运算的基本规则，要提升运算符的优先级可以使用```()```来提升运算符的优先级。

#### 实例

```
1+2*3       # 先执行2*3=6再执行6+1输出7
(1+2)*3     # 使用```()```提升优先级先执行1+2=3，再执行3*3输出9
```
