1. Swoft安装

1.1.1. composer安装

对于composer不清楚的可以查看这里。

componse create-project swoft/swoft project_name

1.1.2. 手动安装

git clone https://github.com/swoft-cloud/swoft
cd swoft
composer install --no-dev # 不安装 dev 依赖会更快一些
cp .env.example .env
vim .env # 根据需要调整启动参数

1.1.3. Docker方式安装

docker run -p 80:80 swoft/swoft

1.1.4. Docker-Compose 安装

git clone https://github.com/swoft-cloud/swoft
cd swoft
docker-compose up

文档连接：https://doc.swoft.org/master/zh-CN/quickstart/install.html
